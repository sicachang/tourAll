# Create- python base環境 (以 `conda python36` 為例)
```cmd
conda create --name myenv python=3.6
```

# install- virtualenv
```cmd
pip install virtualenv
```

# create- virtualenv
```cmd
virtualenv mypython36
```
# move
move created python folder from `./mypython36` to anaconda paht (`..\Anaconda3\envs`)

# activate
```cmd
activate mypython36
```
